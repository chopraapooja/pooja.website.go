# The Go CD static microsite

# Initial setup

```
$ git clone git@github.com:gocd/website.go.git
$ cd website.go
$ ./.init.sh
```


# Development

```
$ ./.init.sh
$ bundle exec middleman server
$ open http://localhost:4567/
```

Open http://localhost:4567 in your browser.
